# Where to Insert Your Chapter 4 Figures

Place your presentation visuals in the Design chapter as follows:

| Your visual | Insert as | Location in chapter |
|-------------|-----------|---------------------|
| **Use case diagram** | Figure 4.1 | After section 4.1.1 (Use Case Diagram) |
| **System architecture diagram** | Figure 4.2 | In section 4.2 (System Architecture), after the first paragraph |
| **SQL / database schema diagram** | Figure 4.3 | In section 4.4 (Data Model and Event Schema), after the first paragraph |

## In Word / Google Docs
1. Add a figure caption below each image: "Figure 4.1 Use case diagram...", etc.
2. Ensure each figure is referenced in the text (the references are already in place).

## File naming suggestion
- `figure-4-1-use-case.png` (or .jpg, .pdf)
- `figure-4-2-system-architecture.png`
- `figure-4-3-database-schema.png`

Save these in a `figures` or `images` folder if your dissertation template uses one.
